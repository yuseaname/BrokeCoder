# Shopify / E-commerce Builder

Hook: Turn browsers into buyers with clean product pages and frictionless checkout.

## What You Do
You implement and optimize Shopify stores. You customize themes, improve product pages, configure apps, and track conversion.

### Daily Work
- Edit themes and sections
- Improve PDPs (images, copy, reviews, FAQs)
- Set up shipping, taxes, and payments
- Track funnel metrics and run experiments

### Deliverables
- Theme customization and component tweaks
- Product page optimization checklist
- App setup (reviews, subscriptions, upsells)
- KPI dashboard and insights

## Who This Is Good For
- Builders who enjoy measurable business impact
- Marketers who like conversion-focused work

## Skills You Need
### Hard Skills
- Shopify themes, sections, Liquid
- PDP best practices and CRO
- App ecosystem and integrations
- Analytics: GA4, Shopify reports

### Soft Skills
- Communication and prioritization
- Iteration and testing mindset
- Reliability and speed

## Tools You’ll Use
- Shopify admin, theme editor, CLI
- GA4, Search Console, heatmaps
- Figma and Docs/Notion for briefs

## Learning Roadmap (0–90 Days)
### 0–30 Days (Foundation)
- Rebuild a demo store with 5 products
- Optimize PDPs with before/after screenshots
- Set up core apps and tracking

### 30–60 Days (Portfolio + Systems)
- Launch a test store; document conversion changes
- Create a reusable PDP template
- Publish 2 case studies (upsells, reviews, trust)

### 60–90 Days (Client-Ready)
- Offer a “Store Optimization” package
- Ship 2 projects; gather testimonials
- Build productized services

## Portfolio Projects (3–6)
- Demo store with clean PDPs
- PDP optimization before/after case study
- Checkout friction reduction project
- App setup + conversion impact study

## How to Get Your First Client/Job
1. Build a demo store with strong PDPs
2. Pitch ecommerce brands with a helpful audit
3. Offer optimization sprints with fixed scope
4. Publish transparent outcomes and wins

## Pricing & Salary Ranges (No Guarantees)
- Employment: $55k–$95k entry; $85k–$130k mid-level
- Freelance: $1,500–$8,000 implementations; retainers: $800–$3,000/mo

> Disclaimer: Store performance depends on product-market fit and traffic. No guarantees.

## Common Mistakes & Fixes
- Cluttered PDPs → Prioritize hierarchy; add proof and FAQs
- Slow themes → Minify assets; limit apps; optimize images
- No tracking → Set up events and dashboards first

## FAQs
- Do I need Liquid? Basic customization helps; start simple.
- Which apps? Reviews, subscriptions, upsells—choose carefully.
- How to price? Fixed scope + performance focus.
