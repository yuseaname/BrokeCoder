# Data Analyst (Entry-Level Remote Track)

Hook: Make data useful—clean, simple, and actionable.

## What You Do
You collect, clean, and analyze data. You build dashboards and share insights with non-technical teams.

### Daily Work
- Write SQL queries and transform data
- Build dashboards and charts
- Document insights and recommendations
- Collaborate with business stakeholders

### Deliverables
- Clean datasets and queries
- Dashboards (e.g., sales, marketing, operations)
- Reports and presentations
- Data dictionary and SOPs

## Who This Is Good For
- Analytical thinkers who enjoy clarity
- Communicators comfortable with data stories

## Skills You Need
### Hard Skills
- SQL and spreadsheet mastery
- Basic statistics and chart selection
- Dashboard tools (Tableau, Power BI, Looker)

### Soft Skills
- Communication and empathy
- Curiosity and problem-solving
- Organization and documentation

## Tools You’ll Use
- SQL (PostgreSQL/MySQL)
- Spreadsheets (Google Sheets/Excel)
- BI tools and visualization

## Learning Roadmap (0–90 Days)
### 0–30 Days (Foundation)
- Learn SELECT, JOIN, GROUP BY
- Clean sample datasets; build simple charts
- Tell a story with a one-page report

### 30–60 Days (Portfolio + Systems)
- Build 3 dashboards (sales, marketing, ops)
- Publish case study with business impact
- Improve speed and data hygiene

### 60–90 Days (Job-Ready)
- Assemble portfolio; request mock interviews
- Apply broadly with proof projects
- Practice business communication

## Portfolio Projects (3–6)
- Sales dashboard with insights
- Marketing funnel analysis
- Operations metrics and forecasting

## How to Get Your First Role
1. Build portfolio projects with practical data
2. Publish clear write-ups of outcomes
3. Apply with proof; communicate simply

## Compensation (No Guarantees)
- Employment: $55k–$85k entry; $75k–$110k mid-level depending on region

> Disclaimer: Outcomes depend on industry, region, and skills; no guarantees.

## Common Mistakes & Fixes
- Overcomplicated charts → Keep it simple and clear
- No documentation → Write data dictionary and context
- Ignoring stakeholders → Ask questions; align goals

## FAQs
- Do I need advanced math? Basic stats and curiosity go far.
- Remote options? Increasingly common; communication matters.
- How to advance? Learn SQL deeply; add Python later.
