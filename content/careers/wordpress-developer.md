# WordPress Developer (More Technical)

Hook: Ship reliable WordPress implementations—custom, secure, and fast.

## What You Do
You implement and extend WordPress: custom themes, plugins, integrations, and performance.

### Daily Work
- Build themes and custom blocks
- Integrate plugins and APIs
- Optimize performance and security
- Maintain sites and fix bugs

### Deliverables
- Custom theme + component library
- Plugin config and customizations
- Performance + security checklist
- Documentation and handoff guides

## Who This Is Good For
- Builders who like practicality and shipping
- Problem-solvers who enjoy PHP/JS

## Skills You Need
### Hard Skills
- WordPress, PHP, Gutenberg blocks
- HTML/CSS/JS; build tooling
- Security basics, backups, staging
- Performance: caching, images, queries

### Soft Skills
- Client communication and scoping
- Documentation and maintainability
- Iterative improvement mindset

## Tools You’ll Use
- IDE (VS Code) and local dev (Local, Docker)
- WP CLI, Composer, Git
- Perf tools: Query Monitor, Lighthouse

## Learning Roadmap (0–90 Days)
### 0–30 Days (Foundation)
- Build a custom theme from scratch
- Create 5 reusable blocks/components
- Set up Git, WP-CLI, staging

### 30–60 Days (Portfolio + Systems)
- Implement a real site with custom theme
- Write docs + handoff checklist
- Publish performance improvements

### 60–90 Days (Client-Ready)
- Offer a “Clean Implementation” service
- Ship 2 sites; collect testimonials and metrics

## Portfolio Projects (3–6)
- Custom theme + blocks
- Plugin integration with custom code
- Performance case study (speed + Core Web Vitals)
- Security audit and fixes

## How to Get Your First Client/Job
1. Build demo theme + document decisions
2. Pitch 20 small agencies with samples
3. Offer pilot implementation at fixed scope
4. Ship fast, clean, and test thoroughly

## Pricing & Salary Ranges (No Guarantees)
- Employment: $60k–$100k entry; $85k–$130k mid-level
- Freelance implementations: $2,000–$10,000+
- Retainers: $600–$2,500/mo maintenance + improvements

> Disclaimer: Results depend on scope, complexity, and reliability of delivery.

## Common Mistakes & Fixes
- Plugin overload → Favor custom + lean stacks
- Poor performance → Measure; fix queries, caching, images
- No staging/backups → Set up processes before changes

## FAQs
- Do I need design skills? Not required; collaborate with designers.
- Which hosting? Pick reliable vendors; measure performance.
- How to price? Scope-first; avoid hourly; define deliverables.
