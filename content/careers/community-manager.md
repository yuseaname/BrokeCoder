# Community Manager / Discord Moderator

Hook: Create safe, helpful spaces where members want to stay.

## What You Do
You design member experience, moderate conversations, run events, and support the community.

### Daily Work
- Moderate chats and resolve conflicts
- Publish announcements and events
- Support members and answer questions
- Gather feedback and improve guidelines

### Deliverables
- Community guidelines and onboarding
- Moderation SOPs and escalation paths
- Events calendar and recaps
- Member FAQ and resources

## Who This Is Good For
- Empathetic communicators who like people ops
- Organized builders who enjoy systems

## Skills You Need
### Hard Skills
- Platform setup (Discord, Slack)
- Moderation tools and permissions
- Event planning and facilitation
- Documentation and SOPs

### Soft Skills
- Calm conflict resolution
- Fairness and consistency
- Clear communication

## Tools You’ll Use
- Discord, Slack, Notion/Docs
- Event tools (Calendars, Streams)
- Analytics (retention, activity)

## Learning Roadmap (0–90 Days)
### 0–30 Days (Foundation)
- Set guidelines and onboarding
- Run 2 small events; collect feedback
- Build moderation SOPs

### 30–60 Days (Portfolio + Systems)
- Launch member FAQ and resource hub
- Train moderators; define escalation
- Publish monthly metrics

### 60–90 Days (Client-Ready)
- Offer community setup package
- Ship for 2 groups; get testimonials
- Iterate processes and frameworks

## Portfolio Projects (3–6)
- Community setup with docs
- Events program with recordings
- Moderation SOPs + outcomes

## How to Get Your First Client/Job
1. Volunteer/mod at small communities
2. Document outcomes and processes
3. Pitch orgs with setup + moderation package

## Salary & Retainers (No Guarantees)
- Employment: $40k–$75k entry; $60k–$90k mid-level
- Retainers: $500–$2,000/mo depending on scope and size

> Disclaimer: Compensation depends on scope, responsibilities, and org needs.

## Common Mistakes & Fixes
- Vague rules → Write clear guidelines; enforce fairly
- No onboarding → Create welcome flow; explain norms
- Burnout → Rotate moderators; set boundaries

## FAQs
- Is moderation hard? It’s people-first; build fair systems.
- How to handle conflict? Calmly; follow SOPs; escalate when needed.
- Do I need events? Events help retention; start small.
