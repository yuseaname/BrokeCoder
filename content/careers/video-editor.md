# Video Editor (Short-form + YouTube)

Hook: Make ideas watchable—fast hooks, clean pacing, clear story.

## What You Do
You edit videos for creators and brands, focusing on attention capture and retention.

## Day in the Life
- Morning: review briefs, cut rough versions, mark hook options
- Midday: refine pacing, add captions, design thumbnails
- Afternoon: export variants, collect feedback, iterate
- Weekly: retention analysis, hook library updates, SOP improvements

### Daily Work
- Cut footage; tighten pacing; add captions
- Design thumbnails and pattern interrupts
- Edit formats for TikTok/Reels/Shorts/YouTube
- Export and deliver assets; manage feedback cycles

### Deliverables
- Edited videos (shorts + long-form)
- Captioned/subtitled versions
- Thumbnail iterations and A/B tests
- Project files and asset management

Extended deliverables:
- Hook library and pattern interrupts pack
- Sound design options and music beds (licensed)
- Template project files for speed
- Subtitle style guidelines and accessibility checks

## Who This Is Good For
- Visual storytellers with attention to detail
- Creators who enjoy fast iteration

## Skills You Need
### Hard Skills
- Editing tools: Premiere, Final Cut, DaVinci Resolve
- Hook writing and story structure
- Caption/subtitle workflows
- Thumbnail design basics

### Soft Skills
- Speed and consistency
- Communication and feedback management
- Organization and asset handling

## Tools You’ll Use
- Editing suites (Premiere, Resolve)
- Caption tools and templates
- Asset libraries, cloud storage

Tip: create a project template with bins, sequences, and export presets to cut turnaround time.

## Learning Roadmap (0–90 Days)
### 0–30 Days (Foundation)
- Edit 20 short-form pieces; practice hooks
- Build caption templates and a pacing checklist
- Create 10 thumbnails; test designs

### 30–60 Days (Portfolio + Systems)
- Edit 3 YouTube videos with strong watch time
- Publish before/after hook improvements
- Build an editing SOP with timelines

### 60–90 Days (Client-Ready)
- Offer a “Content Sprint” (20 shorts + thumbnails)
- Ship for 2 creators; get testimonials
- Document retention improvements

Add one YouTube case study: hook rewrite + pacing changes improves avg view duration.

## Portfolio Projects (3–6)
- Shorts pack (20 videos + metrics)
- YouTube edit with retention analysis
- Thumbnail A/B testing case study
- Caption templates and speed improvements

Acceptance criteria examples:
- Shorts pack: consistent style; retention 35%+ for 30–60s videos
- YouTube edit: higher avg view duration and CTR on new thumbnail
- Caption templates: readable on mobile; consistent style; improved accessibility

## How to Get Your First Client/Job
1. Edit for yourself or friends; publish metrics
2. Pitch creators with a helpful audit and test edit
3. Offer sprint packages and fast turnaround
4. Ask for referrals and testimonials

### Outreach Scripts
1. "Your first 3 seconds don’t land—I cut a tighter hook variant. Want to see it?"
2. "Thumbnails feel cluttered—here are 3 cleaner designs with test data."
3. "Captions are inconsistent—mobile viewers bounce. I have a style guide ready."

## Pricing & Salary Ranges (No Guarantees)
- Employment: $45k–$75k entry; $65k–$100k mid-level
- Freelance packs: $300–$1,500 short-form bundles; $500–$2,500 YouTube edits

> Disclaimer: Income depends on project scope, speed, and client budgets. No guarantees.

## How You Get Paid
- Employment: salary for ongoing channel support
- Freelance: per‑video or bundle pricing; define revisions
- Productized: sprint packs (e.g., 20 shorts + 4 thumbnails)

Document retention and CTR lifts—creators care about outcomes.

## Common Mistakes & Fixes
- Slow hooks → Front-load value; cut fast
- Messy assets → Use structured folders and SOPs
- Inconsistent subtitles → Build templates; keep style tight

Additional:
- Over-editing → Let the story breathe; avoid distraction
- Ignoring mobile → Design for small screens first

## FAQs
- Which tool to use? Pick one; specialize.
- How to get results? Hook clarity, fast pacing, visual variety.
- How to price? Package-based with clear deliverables.
- How many revisions? 1–2 rounds; define what counts as a revision
- Do I need motion graphics? Basic text and cuts are enough for most shorts
